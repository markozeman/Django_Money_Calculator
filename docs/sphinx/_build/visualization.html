Controller¶

vizualizacija.service.prepare_visual_data(danes, stevilo_dni, data, dropdown)¶

Prepares JSON data

From input data parses all important informations into JSON.

Parameters:
  • danes – date today
  • stevilo_dni – number of days for the time period
  • data – data to parse
  • dropdown – chosen value from dropdown (wallet, bank, together)
Returns:

JSON object, prepared for use in JS
vizualizacija.service.process_list(data)¶

Processes list items and changes them into the right form

Goes through the list and saves data to new list in form [money_status, date]

Parameters:data – list of data in form [plus_or_minus, money_status, date]
Returns:new processsed list
vizualizacija.service.show_visual(request)¶

Shows visual graph

Reads user input and shows visual representation on the graph for chosen time period. It shows how your money balance has changed in this period.

Parameters:request – HTTP request
Returns:JSON, which is used with JS to display the chart
